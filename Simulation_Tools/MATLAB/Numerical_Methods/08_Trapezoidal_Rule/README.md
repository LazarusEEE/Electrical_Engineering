# Trapezoidal Rule
## Overview
The Trapezoidal Rule is a numerical integration technique used to approximate the value of a definite integral by dividing the interval into trapezoids and summing their areas.

## Objective
* Approximate definite integrals numerically.
* Compare numerical and analytical solutions.
* Implement the Trapezoidal Rule using GNU Octave/MATLAB.

## Algorithm
1. Define the integration interval.
2. Divide the interval into equal subintervals.
3. Evaluate the function at each point.
4. Calculate the area of each trapezoid.
5. Sum all trapezoidal areas to obtain the approximation.

## Input
* Function
* Lower limit
* Upper limit
* Number of subintervals

## Output
* Approximate integral
* Numerical error (if exact value is known)

## Applications
* Engineering analysis
* Area estimation
* Physics calculations
* Scientific computing
