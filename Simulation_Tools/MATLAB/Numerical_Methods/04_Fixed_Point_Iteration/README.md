# Fixed Point Iteration Method
## Overview

Fixed Point Iteration is an iterative numerical method that solves equations by transforming them into the form:

`x = g(x)`

## Objective
* Solve nonlinear equations using iteration.
* Understand convergence conditions.
* Analyze numerical stability.

## Algorithm
1. Convert equation into fixed-point form.

2. Select initial value.

3. Calculate:

   `x(n+1) = g(xn)`

4. Continue until convergence.

## Input
* Function `g(x)`
* Initial guess
* Tolerance
* Maximum iterations

## Output
* Approximate solution
* Iteration values
* Error
