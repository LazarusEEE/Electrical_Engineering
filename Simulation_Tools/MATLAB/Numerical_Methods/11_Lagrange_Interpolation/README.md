# Lagrange Interpolation
## Overview
Lagrange Interpolation is a polynomial interpolation technique used to estimate unknown function values from a set of known data points. It constructs a unique polynomial that passes through all given points.

## Objective
* Estimate unknown values using interpolation.
* Construct interpolation polynomials numerically.
* Implement Lagrange Interpolation in GNU Octave/MATLAB.

## Algorithm
1. Define the known data points.
2. Construct Lagrange basis polynomials.
3. Form the interpolation polynomial.
4. Evaluate the polynomial at the required point.

## Input
* Known data points
* Interpolation point

## Output
* Interpolated value
* Interpolation polynomial

## Applications
* Engineering measurements
* Experimental data analysis
* Signal processing
* Computer graphics

## Software
* GNU Octave
* MATLAB Compatible

Develop an understanding of polynomial interpolation and its applications in numerical analysis and engineering.
