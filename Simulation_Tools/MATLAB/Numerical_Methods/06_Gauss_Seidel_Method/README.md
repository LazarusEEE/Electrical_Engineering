# Gauss-Seidel Method
## Overview
The Gauss-Seidel Method is an iterative technique for solving systems of linear equations. Unlike the Jacobi Method, newly calculated values are immediately used in subsequent calculations, resulting in faster convergence for many problems.

## Objective
* Solve linear systems using iterative computation.
* Compare convergence with the Jacobi Method.
* Implement the algorithm in GNU Octave/MATLAB.

## Algorithm
1. Express the system in matrix form.
2. Rearrange equations to isolate each variable.
3. Select an initial approximation.
4. Update each variable using the latest available values.
5. Continue until the stopping criterion is satisfied.

## Input
* Coefficient matrix
* Constant vector
* Initial guess
* Tolerance
* Maximum iterations

## Output
* Solution vector
* Number of iterations
* Approximation error

## Applications
* Power system analysis
* Circuit simulation
* Engineering computations
* Numerical optimization

## Software
* GNU Octave
* MATLAB Compatible

## Learning Outcome
Gain practical experience with iterative linear equation solvers and understand their computational efficiency.
