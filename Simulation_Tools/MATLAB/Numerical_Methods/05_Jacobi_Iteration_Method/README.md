# Jacobi Iteration Method
## Overview
The Jacobi Iteration Method is an iterative numerical technique used to solve systems of linear equations.
Each variable is updated using values from the previous iteration, making the method simple to implement and suitable for diagonally dominant matrices.

## Objective
* Solve systems of linear equations iteratively.
* Understand convergence criteria for the Jacobi Method.
* Implement the algorithm using GNU Octave/MATLAB.

## Algorithm
1. Arrange the equations in matrix form.
2. Rearrange each equation to isolate one variable.
3. Choose an initial approximation.
4. Compute new values using only the previous iteration.
5. Repeat until the desired tolerance is achieved.

## Input
* Coefficient matrix
* Constant vector
* Initial guess
* Tolerance
* Maximum iterations

## Output
* Solution vector
* Number of iterations
* Approximation error
