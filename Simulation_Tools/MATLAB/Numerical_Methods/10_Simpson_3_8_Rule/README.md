# Simpson's 3/8 Rule
## Overview
Simpson's 3/8 Rule is a numerical integration technique that approximates the area under a curve using cubic polynomial interpolation. It is particularly useful when the number of subintervals is a multiple of three.

## Objective
* Compute definite integrals using Simpson's 3/8 Rule.
* Study the effect of higher-order interpolation on numerical accuracy.
* Implement the method in GNU Octave/MATLAB.

## Algorithm
1. Define the function and integration interval.
2. Divide the interval into a multiple of three subintervals.
3. Evaluate the function at each point.
4. Apply Simpson's 3/8 formula.
5. Calculate the approximate integral.

## Input
* Function
* Lower limit
* Upper limit
* Number of subintervals (multiple of 3)

## Output
* Approximate integral
* Numerical error
