# Euler Method for Ordinary Differential Equations (ODE)
## Overview
The Euler Method is a first-order numerical technique used to approximate the solution of ordinary differential equations (ODEs) with given initial conditions. It estimates the solution by advancing step-by-step using the derivative at the current point.

## Objective
* Solve first-order ordinary differential equations numerically.
* Understand numerical approximation of initial value problems.
* Implement Euler's Method using GNU Octave/MATLAB.

## Algorithm
1. Define the differential equation.
2. Specify the initial condition.
3. Choose the step size.
4. Compute the next solution using Euler's formula.
5. Repeat until the desired interval is covered.

## Input
* Differential equation
* Initial condition
* Step size
* Final value of the independent variable

## Output
* Approximate numerical solution
* Solution table
* Graph of the numerical solution

## Applications
* Engineering system analysis
* Population models
* Electrical circuit modeling
* Dynamic system simulation

## Software
* GNU Octave
* MATLAB Compatible

## Learning Outcome
Understand the fundamentals of numerical solutions for ordinary differential equations and evaluate the effect of step size on solution accuracy.
