# Runge-Kutta 4th Order Method (RK4)
## Overview
The Fourth-Order Runge-Kutta (RK4) Method is one of the most widely used numerical techniques for solving ordinary differential equations. It provides high accuracy by calculating four slope estimates at each step and combining them into a weighted average.

## Objective
* Solve first-order ordinary differential equations with high accuracy.
* Implement the Fourth-Order Runge-Kutta algorithm.
* Compare RK4 with Euler and Modified Euler methods.
## Algorithm

1. Define the differential equation.
2. Specify the initial condition.
3. Choose the step size.
4. Compute the four intermediate slopes (`k₁`, `k₂`, `k₃`, and `k₄`).
5. Calculate the weighted average of the slopes.
6. Compute the next solution point.
7. Repeat until the required interval is completed.

## Input
* Differential equation
* Initial condition
* Step size
* Final value of the independent variable

## Output

* Solution table
* Graphical representation
* Approximation error
