# MATLAB / GNU Octave Numerical Methods
This folder contains numerical analysis experiments implemented using **GNU Octave** and MATLAB-compatible code as part of my Electrical and Electronic Engineering academic sessional practical work.

## Software Used
- GNU Octave
- MATLAB (Compatible)

## Topics Covered

### Root Finding Methods
- Bisection Method
- False Position Method (Regula Falsi)
- Newton-Raphson Method
- Fixed Point Iteration Method

### Linear Equation Methods
- Jacobi Iteration Method
- Gauss-Seidel Method

### Curve Fitting
- Polynomial Curve Fitting

### Numerical Integration
- Trapezoidal Rule
- Simpson's 1/3 Rule
- Simpson's 3/8 Rule

### Interpolation
- Lagrange Interpolation
- Newton Forward Interpolation
- Newton Backward Interpolation

### Numerical Solution of ODE
- Euler Method
- Modified Euler Method (Heun/RK2)
- Runge-Kutta 4th Order Method (RK4)

## Purpose
- Implement numerical algorithms using Octave/MATLAB
- Understand computational methods for engineering problems
- Analyze numerical accuracy and errors
- Improve programming and mathematical modeling skills
