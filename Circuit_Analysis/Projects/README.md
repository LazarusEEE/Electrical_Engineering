LED Resistor Circuit
Overview

This project demonstrates how to connect an LED with a current-limiting resistor to a DC power supply. It is a beginner-friendly circuit that introduces the basic concepts of voltage, current, and resistance.

Objectives
Learn how to connect an LED safely.
Calculate the required resistor value using Ohm's Law.
Create the circuit schematic using KiCad.
Components
Component	Quantity
LED	1
220 Ω Resistor	1
#DC Power Supply (5 V)	1

Circuit Description

The resistor limits the current flowing through the LED, preventing damage. The LED is connected in series with the resistor and powered by a 5 V DC supply.

##Formula

Current through the LED is determined using Ohm's Law:
R = (Vs − Vf) / I
Where:
Vs = Supply voltage
Vf = LED forward voltage
I = Desired LED current
Learning Outcomes

##After completing this project, you should be able to:
Understand why LEDs require current-limiting resistors.
Apply Ohm's Law to select resistor values.
